What is Data Science?

To put it simply, data science is the study of data and the methods of recording, storing, and analyzing data to extract useful information. A data scientist has more programming knowledge than that of a statistician and more statistical knowledge than that of a programmer, while also using business skills to apply findings to the industry.

What kind of analytical tools are you familiar with?

There are many different tools that can be used to achieve the same purpose. I'm familiar with data science modeling tools in Python, R and SAS, with my personal favorite being Python due to it's ease of use and powerful libraries for machine learning and natural language processing.

What other programming languages are you proficient in?

A lot. For a full list of languages see my resume below, but I've reached the point where learning a new Object-Oriented or Scripting language only comes down to learning the syntax.

Will you build me a new website?

No. While I'm able to do front end design and development my strengths are in backend development and database administration.

Can you build me graphs and models for my website derived from a relational database?

Yes, now you're getting the idea.
